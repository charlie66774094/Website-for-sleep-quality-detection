# 睡眠质量分析工具

## 项目简介

睡眠质量分析工具是一个专业的Web应用程序，旨在帮助用户科学管理睡眠，优化休息质量。该工具基于睡眠周期理论（每个周期约90分钟），提供智能化的睡眠时间计算和质量分析功能。

## 核心功能

### 1. 睡眠时间计算
- 根据用户输入的入睡和起床时间，精确计算总睡眠时长
- 分析完整的睡眠周期数量（每个周期约90分钟）
- 提供理想的起床时间建议

### 2. 睡眠质量分析
- 评估实际睡眠效率和质量评分
- 估算深睡眠时间占比
- 生成个性化改善建议

### 3. 智能推荐系统
- **根据入睡时间推荐起床时间**：基于完整的睡眠周期理论，推荐最佳起床时间
- **根据起床时间推荐入睡时间**：反向计算合适的入睡时间，确保获得充足休息

## 技术特点

- **响应式设计**：完美适配手机、平板和电脑等多种设备
- **科学算法**：基于睡眠医学研究的周期理论
- **直观界面**：简洁明了的用户界面，操作简单便捷
- **实时分析**：即时计算和展示分析结果

## 使用方法

1. 访问主页(index.html)了解工具功能
2. 点击"开始使用"进入睡眠分析工具(1.html)
3. 选择相应功能标签（睡眠时间计算/睡眠质量分析）
4. 输入相关时间信息
5. 查看详细分析结果和改进建议
